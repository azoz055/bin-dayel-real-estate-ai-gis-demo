# ملاحظات عرض منصة عبدالعزيز بن دايل - بن دايل هويتي

## رابط التشغيل المحلي

بعد تشغيل الأمر:

```bash
npm run preview
```

افتح:

http://localhost:4173

وللمشاركة داخل نفس الشبكة، افتح الرابط باستخدام IP الجهاز بدلاً من localhost:

http://YOUR_DEVICE_IP:4173

## رسالة العرض للإدارة

هذا دمو احترافي يستخدم بيانات سعودية تجريبية حول الرياض لإثبات فكرة منصة داخلية للسجل العقاري:

- خريطة تفاعلية لقطع الأراضي.
- طبقات تنظيمية ومخاطر ديمو.
- تحليل فوري عند تحديد قطعة.
- مساعد AI عربي باسم "بن دايل".
- تقرير تنفيذي قابل للطباعة/الحفظ PDF.
- تصميم متجاوب للجوال.

## التوضيح المهم

البيانات ليست بيانات سجل عقاري حقيقية. في النسخة الإنتاجية داخل المنظومة المغلقة يتم الربط مع:

- ArcGIS Enterprise / Portal.
- ArcGIS Server Feature Services.
- ArcPy Workers للتحليل الرسمي.
- Local LLM داخل الشبكة.
- صلاحيات Active Directory وAudit Logs.

## طريقة التشغيل من الصفر

```bash
npm install
npm run build
npm run preview
```
