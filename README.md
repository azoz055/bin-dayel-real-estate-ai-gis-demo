# Bin Dayel Real Estate AI GIS Demo

منصة دمو احترافية باسم عبدالعزيز بن دايل لعرض فكرة: خريطة طبقات عقارية + تحليل أرض + مساعد AI عربي باسم "بن دايل".

## ملاحظات مهمة

- البيانات ديمو/مصطنعة حول الرياض وليست بيانات سجل عقاري حقيقية.
- التحليل يعمل داخل المتصفح بدون إرسال البيانات لأي API خارجي.
- الخريطة تستخدم OpenStreetMap tiles؛ عند التشغيل بدون إنترنت تبقى القطع والتحليل يعملان، لكن الخريطة الخلفية قد لا تظهر.
- التصميم قابل للتحويل لاحقاً إلى ArcGIS Enterprise + ArcPy داخل منظومة مغلقة.

## التشغيل

```bash
npm install
npm run dev
```

ثم افتح:

http://localhost:5173

## بناء نسخة قابلة للمشاركة

```bash
npm run build
npm run preview
```

أرسل مجلد `dist` أو ارفعه على أي استضافة داخلية Static Hosting.
